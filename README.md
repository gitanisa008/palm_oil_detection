#  Palm Oil Detection Using Deep Learning

A deep learning project for detecting palm oil trees using **YOLOv8**.  
This repository contains a Jupyter Notebook for training a custom YOLOv8 model on palm oil tree datasets, covering data preparation, model training, and evaluation.

##  Notebook
- `train_palm_yolov8.ipynb` — end-to-end training pipeline with YOLOv8

##  Tech Stack
- [YOLOv8 (Ultralytics)](https://github.com/ultralytics/ultralytics)
- Python / Jupyter Notebook
- Google Colab (recommended)

## 🚀Getting Started
1. Open `train_palm_yolov8.ipynb` in Google Colab.
2. Connect your dataset (e.g., from Roboflow or Google Drive).
3. Run all cells to train and evaluate the model.
